# 一、综述

在公历的2025年元旦的时候，由于我忙于期末复习，所以我觉得并没有让我进行“年度总结”的机会。对我来说，更加自然于年度总结的时刻是农历新年的除夕日。

这一年对我而言是秩序开始替代漂浮的一年。今年似乎像是我的第二个身心高度发展的高峰（第一个是在中学阶段的青春期），在这一年，我逐渐地走向成熟，而尚未走向成熟。在2025的这一年，我对于大学学习和生活的掌控力逐渐增强。在大一的时候，我仍然觉得有很多东西可以探索，而步入大二，我的目标逐渐明晰，工作和学习逐渐收束稳定。同时，学业的压力也不断变大，但是我觉得我对于学业的把控处于很好的状态。

这一年对我来说是内在精神高度发展的一年。我逐渐意识到，我必须进行生命哲学的思考，否则我会持续性地掉入无意义的深渊，即便我只是在正常地生活和学习。

我逐渐形成了系统分析自身发展的模式——多维理想人格形象。我对于自身发展不再是模糊和直觉驱动的，而是能够研究和分析的。这个理想人格形象经过非常多的调整，目前五个最主要的板块是：身体、专业、语言、艺术、人际。

那么，我们现在开始回顾吧。

# 二、理想人格回顾

## **1. 身体素质与外在形象**

在这一年，我真正意识到我可以有计划地改变自己的身体和形象，这是意识上重大的改变。

我从大一下学期（3月初）开始健身，并学习了各种健身动作，训练计划逐渐成熟，并且目前的负重保持稳定阶段。

我的身高维持在 182cm，体重持续在60~65kg上下浮动。尽管我做了很多主观能动的尝试，似乎效果没有特别好，可能是因为没有长期坚持的缘故。下一年需要继续改进。至于身体外形，我可以很清楚地看到我形成了比较明显的肌肉，这是非常明显的可喜改变。

在有氧耐力方面，我的训练方式主要由每学期的长跑月驱动。训练的强度主要是3km左右的中长跑训练。由于我本身体型较瘦的缘故，跑步对我来说不算特别困难的事情。不过我的水平持续性地处于停滞状态，我没有刻意去提升（主要精力花在无氧训练上）。另外，我这一年没有进行任何游泳训练，下一年可以接触一下。

在体测分数方面，我的成绩有了很大的增长，特别是BMI、引体向上、短跑、1000m长跑，说明我的基本身体素质在逐渐提升。

在日常作息上，我基本上每天在8~9点起床，0点左右睡觉。我不敢说这是最健康的作息方式，但是我觉得这是环境制约和个人习惯共同造成的结果，我似乎只能适应它。同时经常会有熬夜的习惯，特别是大一学年的期末周。希望下一年可以减少晚睡的频率。

在饮食方面，我一开始做了比较激进的方式，有主动性地让自己在午晚餐吃多一些，但是造成了我的饮食十分地循环，并且总是去同一个窗口吃差不多的东西，让我有些不舒适。后面没有刻意地控制自己的食物多样性。我觉得这可能是我的体重没有增长的一个主要原因。

在穿搭方面，我做了很多且（看似）较为激进的尝试。我对于自己的衣服有非常多的更新，我购入了非常多的新衣服。我开始认为，衣服一定要符合自己的性格和思维方式。我形成了理性、克制、偏正式又融合工装风格的穿搭方式，不过我觉得还需要进一步改进，目前来说还是属于较为稚嫩的状态，但是目前的改进已经很令人可喜了。

在发型方面，我开始进行烫发，这也使得我的头发不再软塌，开始能够为我的外在形象加分。下一年可以尝试更多的发型。

在皮肤状态方面，我形成了每天早晚坚持用洗面奶+爽肤水+面霜，并且定期剃须、修甲的习惯。我的面部皮肤状态持续在变好。然而，我的头屑仍然较多，躯干的皮肤仍然容易受到真菌的侵扰，下一年需要继续关注这一问题。

## 2. **专业能力与理性思维**

在大一时期，我失去了对于课堂认真听课学习的信心，开始大规模地进入自学模式，转向高质量的国外CS课程。逐渐地，自学和课堂的冲突越来越大，我曾经陷入非常矛盾的阶段。后来到了大二，我能感受到老师讲课的水平变得很不错（也许是进入了真正专业课的缘故），我开始每节课都比较集中精力听，而不是单纯地自学，这给我减轻了很大压力，并且学习效率也有很大的提升。

在专业课方面，可以说我的掌握程度都比较好，无论是从期末的复习备考，还是最终的成绩来看，都是让我比较满意的。这说明了我的学习方式逐渐科学和稳定，可以继续持续和发展下去。

同时，我建立了个人的博客网站 [zuwei.org](http://zuwei.org)，我在上面完整上传了我大学以来各个课程的笔记，这是比较有标志性意义和长期性意义的事件。我相信这样的行为可以为后来的自己带来很多益处，同时也可以倒逼当下的自己思考进一步走向系统和严谨。

在学术方面，我加入了从事自然语言处理研究方向的导师学习，目前是完成了 CS224N 课程的学习，但是我觉得学习得还不够扎实和严谨，实践很少，缺少稳定的节奏和计划，同时英语阅读让我有些困难，这让我对于科研学习感到很多迷茫。这是我下一年需要重点改进的方向，尽快能够熟悉自然语言处理的相关技术，并形成很好的直觉。在科研的阅读方面，我觉得我现在形成了科学的阅读论文的方法（无论是什么主题），这对于我今后的科研学习是有很大帮助的。

除了核心专业和科研，我对于数学、物理、经济方面的知识感到比较有兴趣，特别是数学，并一度形成了要辅修数学与应用数学的愿望。但是后来经过多方面的权衡（学业、通勤、资金），我最终放弃了这一选择。但是系统学习数学专业知识的愿望仍然没有消失。在大二上学期，我同时选修了解析几何和数学计算方法课程，这似乎是一种弥补（不过也形成了较大压力！）。希望下一年我能怀着更强的主体性去学习更多方面的知识。

在算法竞赛方面，我和两个队友一起参与了非常多的 ICPC/CCPC 比赛，并取得了比较好的成绩。不过实际上，我们并没有进行非常多的训练，更多的只是延续中学时代的思维习惯和知识体系。在大二上寒假，我们参加了 ICPC EC-Final 杭州站的比赛，取得了决赛银奖的奖项，可以说这是我们的最好成绩。我对于下一年是否参加仍然存有疑问，但是可以说，我的算法竞赛生涯确实是即将走到尾声了。

在学习方法方面，在阅读时，我形成了边学习，边做笔记，边输出的形式，这种方式目前来说是最适合我的，我能够及时地处理当前学习到的知识，并结构化输出。同时，这样还形成了比较完整的知识库，可以让我及时地回看和修改。

## 3. **语言能力与世界意识**

在这一年，我的语言能力增长没有太明显。

在英语方面，训练的机会和时间太少，导致我的水平没有太大的增长。听说读写似乎都在缓慢地倒退，唯一可能有增长的也许是词汇量。这一年考了英语四六级，应该都是通过了。但是这远远没有达到我所需的水平，下一年需要重点关注，争取能够提升到比较高的水平。

在其他语言方面，我对于日语和西班牙语产生了比较大的兴趣（特别是西班牙语）。我开始用多邻国学习这两门语言，目前可以阅读比较简单的文本，并进行一些简单的交流（也许是A1~A2之间的水平）。但是我觉得多邻国的学习效率过于低效率，接下来需要探索更加高效率的方式。

## 4. 艺术修养和精神表达

这一年，我对于艺术的感知主要在音乐和阅读上。

在音乐方面，我继续广泛地探索英美摇滚、后朋克、合成器、新浪潮、独立、梦幻音乐等等，没有太大的变化，不过更加广泛和深入了。这一年我最喜欢的歌手是 New Order (Joy Division), The Smiths, Queen, David Bowie, The War On Drugs, The Jesus And Mary Chain 等等。在大二上的期末周，Queen 乐队的歌曲给我了很大的动力，一种称之为生命活力的东西，我觉得这是我目前急需的。可以说，Queen 是我的“年度歌手”。

在文哲阅读上，由于要充分考虑时间利用（大段的课余时间其实比较少），我实行电子书和纸质书相互穿插的形式。这一年，我看了《局外人》《百年孤独》《活出生命的意义》《在轮下》《人生》《地下室手记》《献给阿尔吉侬的花束》《美丽新世界》等等，主要聚焦于生命意义，荒诞哲学，软科幻等等。在后期，我看了《维特根斯坦传》，我似乎在这位哲学家的人身轨迹上找到了很强烈的精神共鸣，维特根斯坦的精神姿态让我非常佩服和欣赏，这是我的“年度偶像”。

在游戏方面，虽然我不是经常玩游戏，但是我的确开始在逐渐探索主机游戏。这一年让我感触和共鸣最深的是《极乐迪斯科》这款游戏。这款游戏的存在姿态、哲学思考、人性选择让我非常地吸引，并改变了我一部分对于生活的看法。毫无疑问，这是我的“年度游戏”。

目前，我的艺术感知大多只是停留于感性体验，真正主动写下来的评论和感悟很少或者很碎片化。但是我希望自己可以逐步开始写文艺长评，并逐渐上传到我的个人博客网站上，形成独属于自己的一份文艺空间和精神姿态。

在这一年，也许是生活和学习压力过于重的缘故，我陷入了大量的哲学思考，思考的对象主要是有关意义、自我本质、个体主权、机器理性等等，这些东西似乎是自发流动到我的脑海中，我不得不去思考这些。我似乎处于精神的快速扩张和重组的阶段，由此形成了我对于哲学的巨大需求，同时也造成了比较大的精神负载。完全可能的是，下一年我必然会继续思考这些东西，并大量阅读相关的著作。我希望把这种思考稳定在比较健康的水平而不处于极端状态，并能够产出自己的一些独特思考。

## **5. 人际交往与人格魅力**

这一年我的人际交往处于比较低迷的状态，我的朋友极端地少，因此似乎是极端的孤独。我更适应于独自计划和行动。尽管我有很多思考和情感，但我更多是和自己进行交流。同时，我的个人气质也许是比较忧郁和沉静，没有过分张扬。在这一方面，我不知道是否应当维持现状。

对于形成人际关系，我讨厌浅层的交流，这样对我自身的思考和生活节奏会带来很大的破坏，但是我难以形成深度交流，这实在是很矛盾。

# 四、下一年目标

## **1. 身体素质与外在形象**

- 体重达到70kg，努力让肌肉更加匀称和明显。下一年我需要把训练与饮食系统绑定，并逐步提升健身负重，而不是只提高主观意愿。
- 逐渐开始恢复游泳，提升自身的心肺能力
- 继续改进自己的衣橱，达到比较稳定的风格
- 根本性地解决自己的皮肤易感真菌问题

## 2. **专业能力与理性思维**

- 沿用当前方法继续学习专业课知识，努力达到更高的理解体系（顺便获得高绩点）
- 必须建立完整的自然语言处理知识体系，打好坚实的基础
- 至少完成一个完整的科研项目，和导师有稳定、持续的联系
- 博客网站继续完善，成为完整的思考介质
- 至少学习三门数学与应用数学课程，继续补全体系

## 3. **语言能力与世界意识**

- 提升自己的英语水平到完全能支持科研学习的程度，并且听说能力要结构性地提高
- 提升日语、西语到B1~B2水平

## 4. 艺术修养和精神表达

- 写文艺长评，无论是文学还是音乐，至少选10篇自己非常喜欢的作品进行评论。跨越从感性体验到文字理性认知的障碍。
- 阅读至少5本严肃的哲学著作，并做好相关笔记，抒发自己的思考
- 保持文哲持续阅读

## 5. **人际交往与人格魅力**

- 认识多一些朋友
- （不强求）找到真正的知己

# 五、总结

**这一年，我尚未走向成熟，但已经离开了无意识的漂浮状态。** 我逐渐从一个被动适应环境的人，转变为一个能够主动构建自身结构的人。尽管许多问题仍未解决，许多方向仍需深入，但秩序已经建立，轨道已经出现。

那么接下来的一年，我应当将这个结构继续发展和改进，以让我处于身心健康，多维发展的状态。我对下一年充满了十足的信心（虽然不知道这是否盲目）。无需多言，请继续去走你的人生吧。
